# CRM Settings — Сценарии настройки Битрикс24

Интерактивный чеклист для пошаговой настройки CRM Битрикс24. Открывается как обычная HTML-страница прямо в браузере — без сервера, без зависимостей.

## Что внутри

Один файл `index.html` содержит:

- **12 сценариев настройки** с разбивкой по задачам:
  - Запуск CRM с нуля (воронки, справочники, реквизиты)
  - Структура данных (пользовательские поля, свойства товаров, аналитика)
  - Интерфейс и поведение (карточки, виды списков, уведомления)
  - Автоматизация продаж (роботы, бизнес-процессы, повторные продажи)
  - Дела и календарь (счётчики, автозакрытие дел при конвертации)
  - Почта в CRM (email-трекер, шаблоны, правила)
  - Интеграции и каналы (1С, SMS-провайдеры, WhatsApp)
  - Документы и оплата (способы оплаты, нумераторы)
  - Доступы и права
  - Маркетплейс и отраслевые сценарии
  - Контроль качества данных (дубликаты, форматирование)
  - Здоровье CRM (история, корзина, место на диске)

- **Типы шагов:**
  - `link` — ссылка на раздел настроек портала
  - `settings_group` — группа настроек с тоглами, селектами и текстовыми полями
  - `matrix` — матрица (например, автозакрытие дел по типам при конвертации лидов)
  - `group` — сгруппированные ссылки (например, все SMS-провайдеры)

- **Интерактивность:**
  - Прогресс по каждому сценарию сохраняется в `localStorage`
  - Поиск по названиям и шагам
  - Фильтрация по сущностям: лиды, сделки, контакты, компании, счета, предложения
  - Условная видимость настроек (зависимые поля)

## Как использовать

1. Скачайте или откройте `index.html` в браузере
2. Ссылки «Открыть» в шагах ведут на относительные URL вида `/crm/configs/...` — они работают, если открыть страницу в контексте вашего портала Битрикс24, либо использовать как справочник при ручной навигации
3. Галочки и значения настроек сохраняются между сессиями в `localStorage`

crmsettings/

## Назначение

Файл используется для сравнения эталонных значений настроек CRM с реальными значениями на портале — как чеклист при аудите или первичной настройке системы.
